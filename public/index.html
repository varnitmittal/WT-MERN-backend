Welcome to WeightTrackerMERN backend!!

WeightTrackerMERN is a full-stack javascript based webapp, built using
MongoDB, Express, React, Node (MERN stack).
It's a simple fitness application that helps users to keep track of
their body-weight and BMI with the help of appropriate visual representation of the data.

CRUD Api endpoints:
  • signup
  • login
  • profile
  • record



signup

endpoint: /signup
Used for registration of new users(accounts) to the application.


login

endpoint: /login
Used for logging into account(s).


profile

endpoint: /profile
sub-endpoints: /create, /read, /update, /delete
Meant for managing user's profiles.


record

endpoint: /record
sub-endpoints: /create, /readOne, /readAll, /updateOne, /deleteOne, /deleteAll
Meant for managing user's (health) records.
